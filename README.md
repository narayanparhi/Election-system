# Election-system
This project reveals the use of structures, conditional statements, loops, arrays, file handling, and functions etc. 

UNIQUENESS OF ELECTION SYSTEM PROJECT :-

1. Pre-Configured Candidates with Taglines

Your project initializes candidates with:

Unique IDs

Full names

Taglines in parentheses:

Alice (The Builder)

Bob (The Visionary)

Charlie (The Reformer)

2. Strong Input Validation for Voting

Your system detects and handles:

Non-numeric input

Invalid candidate IDs

Automatically clears input buffer

This prevents crashes and enhances robustness.

FEATURES OF ELECTION SYSTEM PROJECT :-

1. Candidate Setup

Automatically loads 3 predefined candidates

Sets initial vote counts to 0

Supports up to MAX_CANDIDATES = 5

2. Voting Module

Displays all candidates with names & IDs

Accepts numeric input for candidate ID

Validates input (no crash on text input)

Updates vote count for the selected candidate


Informs the user of voting success
